# Changelog

## v3.0.0

### Переписывание на Go
- Полный переход с Python (FastAPI) на **Go (Fiber v2)**
- Замена `aiosqlite` на `modernc.org/sqlite` (чистый Go SQLite)
- Замена `paramiko` на `golang.org/x/crypto/ssh` + `pkg/sftp`
- Замена `python-telegram-bot` на `telebot.v3`
- Генерация кода БД через **sqlc**
- Многоэтапный Dockerfile на базе `golang:alpine`

## v2.2.0

### Исправления стабильности
- Критическое исправление Telegram-бота — исправлен баг с асинхронной обработкой callback-запросов.

### Оптимизация производительности
- Оптимизация SQLite — добавлен индекс `idx_users_username`.
- Устранение блокировок Event Loop — хеширование паролей переведено в `asyncio.to_thread`.
- Агрегация SSH-запросов — метрики собираются одной транзакцией вместо 5 последовательных вызовов.
- Параллельный фоновый опрос — статистика трафика собирается через `asyncio.gather`.
- Интеграция TTL-кэша — списки клиентов кэшируются на 15 секунд.

## v2.1.0

### Новые возможности
- SQLite вместо JSON — данные хранятся в `panel.db` с поддержкой транзакций.
- Полная асинхронность через `aiosqlite`.
- TTL-кеш для данных пользователей, серверов и SSH-запросов.

### Удалено
- Импорт пользователей (Remnawave и другие источники).
- `data.json` — заменён на SQLite.
